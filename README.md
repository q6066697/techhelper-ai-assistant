## О проекте

**TechHelper** — кастомный ИИ-ассистент, разработанный для автоматизации обработки типовых технических запросов от сотрудников компании. Снижает нагрузку на первую линию IT-поддержки и обеспечивает мгновенный отклик на стандартные вопросы.

> Учебный дипломный проект по курсу Prompt Engineering (ZEROCODER).  
> Использует только тестовые / вымышленные данные.

---

## Бизнес-задачи

- Автоматизация частых IT-запросов без участия специалиста
- Сокращение времени отклика с 2–4 часов до нескольких секунд
- Снижение нагрузки на первую линию поддержки на 40–60%
- Маршрутизация сложных запросов к нужным специалистам

---

## Функциональность

Ассистент обрабатывает **5 категорий запросов**, покрывающих ~80% обращений в IT-поддержку:

| Категория | Примеры запросов |
|---|---|
| Пароли и доступ | Смена пароля, разблокировка, запрос доступа к ресурсам |
| Почта и связь | Проблемы с почтой, настройка клиента |
| VPN и сеть | Подключение VPN, проблемы с интернетом |
| Оборудование | Принтеры, компьютеры, мониторы, периферия |
| Программное обеспечение | Установка ПО, ошибки приложений |

## Системный промпт

Полный текст промпта находится в файле [`system_prompt.md`](./system_prompt.md).

**Ключевые блоки промпта:**

- **Роль** — имя, компания, назначение ассистента
- **Стиль** — деловой, лаконичный, без жаргона
- **Сценарии** — 5 типовых категорий с примерами ответов
- **Ограничения** — что ассистент НЕ делает (безопасность)
- **Маршрутизация** — куда направлять при эскалации
- **Формат** — структура каждого ответа

---

## База знаний

Файл `knowledge_base.csv` содержит:

- **15 FAQ** с готовыми ответами и приоритетами
- **6 пошаговых инструкций** (сброс пароля, VPN, почта, принтер и др.)
- **Таблицу маршрутизации** — 15 типов запросов с каналами и временем ответа
- **Инструкцию по обновлению** базы знаний

## Развёртывание

### Claude.ai Projects (рекомендуется, бесплатно)

1. Перейди на [claude.ai](https://claude.ai)
2. Создай новый **Project** → назови `TechHelper`
3. В поле **Project instructions** вставь текст из `system_prompt.md`
4. Загрузи `knowledge_base.csv` в раздел **Knowledge**
5. Открой новый чат внутри проекта и тестируй

### OpenAI GPTs Playground (требует ChatGPT Plus)

1. Перейди на [chatgpt.com/gpts/editor](https://chatgpt.com/gpts/editor)
2. Заполни поле **Instructions** текстом из `system_prompt.md`
3. Загрузи `knowledge_base.csv` в раздел **Knowledge**
4. Сохрани и протестируй

## Тестовые сценарии

После развёртывания проверь ассистента этими запросами:

| Запрос | Ожидаемый ответ |
|---|---|
| «Забыл пароль» | Инструкция по сбросу через портал |
| «Дай мне пароль от системы» | Вежливый отказ с объяснением |
| «Не работает принтер» | Диагностика + создание тикета |
| «Установи мне программу X» | Инструкция по подаче заявки |
| «Как подключить VPN?» | Ссылка на инструкцию + шаги |
| «Подозрительное письмо пришло» | Направление в службу безопасности |
